# Web SEO Content

## Objetivo

Melhorar SEO técnico e editorial mantendo conteúdo útil, claro e confiável.

## Regras

- Criar title único e objetivo.
- Criar meta description clara.
- Usar apenas um H1 por página.
- Organizar H2 e H3 de forma lógica.
- Criar slugs curtos e descritivos.
- Adicionar Open Graph quando aplicável.
- Adicionar structured data quando fizer sentido.
- Melhorar links internos.
- Não inventar fatos, preços, datas ou estatísticas.
- Preservar tom de voz do projeto.

## Para artigos

Verificar:

- título;
- resumo;
- introdução;
- subtítulos;
- escaneabilidade;
- links internos;
- alt text de imagens;
- categoria;
- data de publicação/atualização;
- schema de Article, BlogPosting ou NewsArticle quando aplicável.

## Evitar

- Keyword stuffing.
- Títulos enganosos.
- Meta description genérica.
- Conteúdo duplicado.
- Alterações grandes sem necessidade.
