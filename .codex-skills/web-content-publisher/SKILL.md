# Web Content Publisher

## Objetivo

Criar e revisar conteúdo web com clareza, organização editorial e boa estrutura para leitura.

## Regras

- Manter linguagem clara e acessível.
- Adaptar o tom ao público do projeto.
- Criar título objetivo.
- Usar introdução curta.
- Organizar conteúdo com subtítulos.
- Evitar parágrafos longos.
- Sugerir links internos.
- Não inventar fatos recentes sem fonte.
- Revisar ortografia e consistência.

## Estrutura recomendada

1. Título.
2. Resumo curto.
3. Introdução.
4. Desenvolvimento com subtítulos.
5. Conclusão ou chamada final.
6. Categoria.
7. Sugestão de slug.
8. Meta description.
9. Alt text da imagem principal.

## Evitar

- Conteúdo genérico demais.
- Texto muito técnico para público iniciante.
- Clickbait enganoso.
- Informações sem fonte.
